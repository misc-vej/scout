    Read the existing file. The current filter+map chain at the bottom is:
    ```
    const filtered: SpeciesResult[] = results
      .filter(row => { /* exclude locked */ })
      .map(({ seasonLockStart: _s, seasonLockEnd: _e, ...rest }) => rest);
    ```

    Replace this entire block with a single .map() that includes all species but flags locked ones:
    ```
    const filtered: SpeciesResult[] = results.map((row) => {
      const { seasonLockStart: _s, seasonLockEnd, ...rest } = row;

      let isSeasonLocked = false;
      if (row.sensitivityLevel === 'restricted' && row.seasonLockStart && seasonLockEnd) {
        isSeasonLocked = mmdd >= row.seasonLockStart && mmdd <= seasonLockEnd;
      }

      return {
        ...rest,
        isSeasonLocked,
        seasonUnlocksAt: isSeasonLocked ? seasonLockEnd : null,
      };
    });
    ```

    The `mmdd` variable is already defined higher in the function. The `results` query already selects seasonLockStart and seasonLockEnd from the DB — do not change the SELECT.

    Season-lock only applies to `sensitivityLevel === 'restricted'` species. All other species get `isSeasonLocked: false` and `seasonUnlocksAt: null`. (per D-15)

    function formatMMDD(mmdd: string): string {
      const [month, day] = mmdd.split('-').map(Number);
      return new Date(2000, month - 1, day).toLocaleDateString('en-GB', { month: 'short', day: 'numeric' });
    }
